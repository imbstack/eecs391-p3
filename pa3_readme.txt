Descriptions for each java file:

1. Agent.java: 
This interface defines functions agents need to implement.

2. DemoAgent.java: 
An implementation of an agent that moves randomly in the maze.

3. DrawClass.java: 
This class draws the game on the screen.

4. Game.java: 
This file contains 4 classes as follows:
Game: The environment of the whole game.
Location: The class Location is used as data structure for storing the location information of Agent or enemy.
Reward: As a datastructure to store some temporary data.
Wall: As a datastructure to store some temporary data.

5. RL.java:
The reinforcement learning main loop.
